Мінімум:

Задача 1:

Виконай додавання 0,1 і 0,2 добийся математично правильної відповіді.


Задача 2:

Задача 3:

Користувач вказує обсяг флешки в Гб. Програма повинна порахувати, скільки файлів розміром 820 Мб поміщається на флешку.


Норма:

Задача 1:
