Если вы нажмете кнопку «Отправить», данные формы будут отправлены на страницу под названием "/action_page.php".

8.Input Type Color

input type="color" используется для полей ввода, которые должны содержать цвет. В зависимости от поддержки браузера в поле ввода может отображаться палитра цветов.

9.Input Type Date

input type="date"> используется для полей ввода, которые должны содержать дату. В зависимости от поддержки браузера в поле ввода может отображаться средство выбора даты.

10. Тип input Datetime-local

input type="datetime-local"> указывает поле ввода даты и времени без часового пояса. В зависимости от поддержки браузера в поле ввода может отображаться средство выбора даты.

11. Input Type Image

Тип ввода Изображение input type="image"> определяет изображение как кнопку отправки. Путь к изображению указывается в атрибуте src.

input type="email"> используется для полей ввода, которые должны содержать адрес электронной почты. 13. Input Type File - определяет поле выбора файла и кнопку «Обзор» для загрузки файлов.
14.Input Type Hidden
input type="hidden" определяет скрытое поле ввода (не видимое пользователю). Скрытое поле позволяет веб-разработчикам включать данные, которые пользователи не могут видеть или изменять при отправке формы. Скрытое поле часто хранит запись базы данных, которую необходимо обновить при отправке формы. Примечание. Хотя значение не отображается пользователю в содержимом страницы, оно видно (и может быть отредактировано) с помощью инструментов разработчика любого браузера или функции «Просмотр исходного кода». Не используйте скрытые входы в качестве меры безопасности!
